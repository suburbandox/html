The Daiquiri

The Daiquiri family consists of cocktails containing a base spirit (classically rum), a citrus (classically lime juice) and a sweetener. It brings to mind the beach, resort life, and a relaxed attitude. On the beach (or walking down Bourbon Street in New Orleans), one might find adult slushies, with the elements above blenderized with ice. The daiquiri can be much more sophisticated, as can be seen in this section's recipes.
Many cocktails in this class are created with an egg-white foam. [Vegans can use aquafaba, the liquid in a can of chickpeas, which amazingly foams up almost as well as egg whites] The key to a successful foam is a dry shake, in which all the cocktail's ingredients, including the egg white but EXCLUDING the ice, are added to the shaker, which is then vigorously shaken for 30-60 seconds. Next, the ice is added, and is shaken again. Pouring into a coupe glass will provide a cocktail with a creamy and inviting foam topper.
